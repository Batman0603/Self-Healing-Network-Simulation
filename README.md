## 🧠 Overview

This project simulates a **self-healing distributed network** where failures are not disasters… just events.

Powered by:
- 🗣️ Gossip Protocol  
- ❤️ Heartbeat Monitoring  
- 🤖 AI-driven anomaly detection  
- 🔁 Dynamic rerouting

---

## 🔥 Why This Project Hits Different

✨ **Decentralized by design**  
No single point of failure. Nodes share truth like a distributed brain.

🧠 **AI that actually does something useful**  
Predicts failures before they happen. Not just “AI-powered” for vibes.

⚡ **Real-time everything**  
Live updates via WebSockets. No refresh. No patience required.

🔁 **Self-healing network**  
Traffic reroutes automatically when nodes go down.

## 🧩 Core Services

| Service | Role |
|--------|------|
| 🚪 API Gateway | Routes all communication |
| 🗣️ Protocol Service | Gossip + heartbeat consensus |
| 🤖 AI Service | Anomaly detection & prediction |
| 🗺️ Network Service | Graph + routing logic |
| 💥 Simulation Service | Generates failures |
| 📊 Metrics Service | Aggregates health data |
| ⚡ WebSocket Service | Real-time updates |
